## Archived S8 Artifacts

The zip files in this folder store the large generated artifacts for S8:

- `S8-runs.zip`
- `S8-models-output.zip`
- `S8-evaluation-results.zip`

They are tracked in Git instead of the expanded raw directories to reduce clone
size and make GitHub browsing more manageable.

Restore them from `Datasets/sublist/S8-Model-driven Automation/` with:

```bash
unzip archives/S8-runs.zip
unzip archives/S8-models-output.zip
unzip archives/S8-evaluation-results.zip
```
