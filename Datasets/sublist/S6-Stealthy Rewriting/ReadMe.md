# S6 Stealthy Rewriting

The S6 stealthy rewriting datasets are not publicly released.

This stage studies rewriting, paraphrasing, and adaptation of phishing content
for fluency, contextualization, or detector-evasion analysis. The processed
datasets and generated rewriting artifacts may contain operational phishing
content or prompts that could increase misuse risk if distributed publicly.

For ethical and safety reasons, the S6 data files and generated artifacts are
withheld from the public repository. This directory intentionally keeps only
this notice and the upstream fuzzer dataset reference.
