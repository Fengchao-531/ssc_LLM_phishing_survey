# Industry Detector Tables

The left half is a single `HW / GD` overall summary for the detector.
This is computed by concatenating all available HW rows first, then recomputing the metrics once.
The right half keeps stage-level `LLM` metrics.
Metric order is `precision`, `recall`, `accuracy`, `fpr`, `fnr`, `f0_5`, `f1`, `f2`, `mcc`.
`-` means the detector-stage-source combination does not currently have usable predictions.
