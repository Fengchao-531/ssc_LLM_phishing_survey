# Table 2: Stage-Level LLM Difficulty

This export is split into two parts:

- A top-level `MCC` total table that shows each detector's `LLM-P MCC`, `HW overall MCC`, each `LLM stage MCC`, and the `LLM-P +Head MCC` difference.
- A stage-level transfer table below using `Precision`, `Recall`, and `Specificity`.

MCC overview CSV exports:

- `All detectors`: `mcc_stage_detector_overview_all.csv`
- `Academic`: `mcc_stage_detector_overview_academic.csv`
- `Industrial`: `mcc_stage_detector_overview_industry.csv`
- `LLM-P vs LLM-P +Head Recall`: `recall_llm_p_vs_head_comparison.csv`
- `Detector family comparison`: `detector_family_average_comparison.csv`

In all MCC tables:

- `LLM-P MCC` is the detector's aggregate score on the LLM phishing evaluation set.
- `HW Overall MCC` is the detector's aggregate score on the GD/HW evaluation set.
- `llm_*` columns in the CSVs, and the stage block in the HTML tables, are the detector's MCC on each LLM stage.
- The standalone `LLM-P vs LLM-P +Head` comparison table uses `Recall`, not MCC.
- `LLM-P +Head Recall` comes from the header-aware 203-email ablation set, when available.
- `Difference = (LLM-P +Head Recall) - (LLM-P Recall)`.

Stage transfer table rules:

- `HW` is the fixed detector-family `GD/HW overall` mean for that metric, so it stays constant across stages within the same family.
- `LLM` is the mean `LLM stage` value across the detectors that have usable stage results for that stage and metric.
- `Δ = LLM - HW`, computed from the aggregated `LLM` and `HW` values shown in the table.
- `Precision`, `Recall`, and `Specificity` are shown as percentages (`value * 100`) with two decimal places.
- `Specificity = 1 - FPR`, so all three main metrics are aligned as `higher is better`.
- An additional `Precision / Recall / FPR` export is also written to `table_2_stage_transfer_trend_fpr.csv`.
- This version keeps `S6` and `S8` split into their sub-stages because their behavior is meaningfully different.
- Blank cells mean that detector family currently has no usable paired values for that stage/metric.
