# Industry Output Placeholder

The unified industry runner writes generated outputs here by default. Public GitHub keeps only this README and the runner script.

Local output folders may include:

- `LLM-Ind/`
- `HW-result/HW-Ind/`
- `logs/`
- temporary chunk directories under `logs/tmp/`

Result CSVs, logs, manifests, and temporary detector inputs are ignored and should not be committed.
