# Open-Source Detector Wrappers

This directory keeps wrapper scripts for third-party open-source detector projects.

Clone the upstream projects locally into these placeholder directories before running the wrappers:

```text
Detectors/Industry/email_detectors/open-source-git/Phishing-Email-Agent/
Detectors/Industry/email_detectors/open-source-git/email-phishing-detection_V3/
```

The cloned third-party repositories are ignored by Git. If your clones live elsewhere, pass `--repo-dir /path/to/repo` to the wrapper or through `run_text_detectors.py`.
